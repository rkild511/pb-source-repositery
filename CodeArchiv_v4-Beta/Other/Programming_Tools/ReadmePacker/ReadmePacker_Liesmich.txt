Standard
========

Dies ist eine Standard-Vorlage um neue Programmen zu erstellen.



RPack
---------------------------------------------------------------------
Dieses Programm soll Liesmich- oder Info-Dateien zusammen packen, damit man sie ins Programm einfügen kann - ist dann nicht mehr so leicht zu manipulieren und spart Platz.

In der _RPack.ini können zwei Werte angegeben werden:
SourceFile: Dies ist die eigentliche Textdatei.
TargetFile: Diese Datei wird von _RPack.exe erzeugt und kann mit dem IncludeBinary-Befehl in den PureBasic-Quellcode integriert werden.

Im Standard-Quellcode ist eine Prozedur enthalten, welche dies wieder entpackt.

Zur allgemeinen Vorgehensweise:
1. Man ändert die Textdatei
2. _RPack.exe ausführen -> der Text wird gepackt
3. Nun kann das eigene Programm neu kompiliert werden - schon ist der Text aktualisiert...
